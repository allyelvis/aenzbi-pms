# **App Name**: Aenzbi PMS

## Core Features:

- Room Catalog: Display available rooms, descriptions and associated photos
- Availability Calendar: Allow viewing a read-only calendar for room availability across the property.
- Rate Display: Display prices for each room or unit on a per-night basis.
- Date-based search: Enable search based on dates for room availability

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) for sophistication and tranquility.
- Background color: Very light lavender (#F5EEF8).
- Accent color: Royal purple (#7B68EE), creating contrast and visual interest.
- Body and headline font: 'Alegreya', a humanist serif with an elegant, intellectual feel.
- Use elegant, minimalist icons for navigation and key features.
- Ensure a clean, intuitive layout that allows for easy navigation and browsing.
- Incorporate subtle transitions and animations to enhance user engagement without distraction.
